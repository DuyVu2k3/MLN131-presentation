# 🇻🇳 Dân Tộc - Lý Luận & Bản Sắc Việt

Website thuyết trình về bản sắc dân tộc Việt Nam dựa trên lý luận Triết học Mác-Lênin.

## 📋 Nội dung

- **Phần 1**: Phân định Dân tộc (Nghĩa hẹp vs Nghĩa rộng)
- **Phần 2**: Cương lĩnh dân tộc của Chủ nghĩa Mác-Lênin
- **Phần 3**: Giải mã tâm lý người Việt
- **Kết luận**: Thông điệp & Suy ngẫm

## 🚀 Deploy lên Vercel

### Cách 1: Deploy từ GitHub (Đề xuất)

1. **Tạo repository trên GitHub**
   - Vào [github.com](https://github.com) và tạo repository mới
   - Copy toàn bộ code vào repository

2. **Deploy trên Vercel**
   - Vào [vercel.com](https://vercel.com)
   - Click "Add New Project"
   - Import repository GitHub của bạn
   - Vercel sẽ tự động detect Vite và setup
   - Click "Deploy" và đợi vài phút

3. **Xong!** Website của bạn sẽ có URL: `https://ten-project.vercel.app`

### Cách 2: Deploy trực tiếp bằng Vercel CLI

```bash
# Cài đặt Vercel CLI
npm i -g vercel

# Deploy
vercel

# Làm theo hướng dẫn trong terminal
```

## 🛠️ Chạy local

```bash
# Cài đặt dependencies
npm install

# Chạy dev server
npm run dev

# Build production
npm run build
```
